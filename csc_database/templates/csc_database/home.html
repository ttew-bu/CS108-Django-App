How to Use the CSC Database Effectively

  • Within this database, there are 3 models that you can interact with: Volunteers, Community Partners, and Service Events.
  • Pages can be navigated using the toolbar above as well as clicking on any partner, event, or volunteer name as those are hyperlinked with the appropriate pages
  • Forms will guide you with how to input information if incorrect, but you can always edit every object for every model using that model's update page
  • Volunteer and partner impact reports generate automatically as events associated with each are completed; if you miss an event, you can always backdate information using the create/update functions

    • Why does the database exist?

    This database exists so that the Community Service Center can more effectively track all of the models above and track the relative impact of the service. Some examples of this include seeing past volunteers at a community partner or having volunteer recommendations for a service event. The database utilzes the concept of "service value" as a different way to track volunteer service beyond the typical benchmark of hours. It is important to know that the service values in the database right now are arbitrary; it should be up to the CSC team to figure out what certain service opportunities are worth. For all external-facing inquiries or information, please use the CSC's regular website.
